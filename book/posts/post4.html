Differential Equations Explained

October 1, 2024 - Written by: hackhubafrica

A differential equation is any equation that contains the derivatives or differentials of one or more dependent variables with respect to one or more independent variables.

Imagine you have a toy car going on a track. Now, if we want to know how fast the car is going, we don't just look at where the car is at one moment—we look at how much it moves between two moments. That's kind of like a "derivative."

A differential equation is like a rule that tells us how things change. Instead of saying exactly where the toy car is, it tells us how fast it's going or how much it speeds up or slows down as time goes by.

So, a differential equation helps us understand how things are changing, like how fast or slow something is moving or growing, instead of just where it is at one point.

Dependent and Independent Variables

Let's first break down what dependent and independent variables mean in the context of a differential equation:

  • Dependent variable: This is the quantity that is changing, and its value depends on the other variables. For example, if we are tracking the speed of a car, the speed is the dependent variable because it changes based on time.
  • Independent variable: This is the quantity that we control or observe as it changes. For example, time is often the independent variable because other things (like position or speed) depend on it.

Examples

Example 1:

Equation: \( \frac{dy}{dt} = 3t \)

Dependent variable: y (this is what changes over time, like the position or speed of the toy car).

Independent variable: t (time is the independent variable here because the changes in y depend on t).

Example 2:

Equation: \( \frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2} \)

Dependent variable: u (this could represent something like the temperature or height that changes over time and space).

Independent variables: t (time) and x (position in space, like where the car is on the track).

Example 3:

Equation: \( \frac{d^2y}{dt^2} = -9.8 \)

Dependent variable: y (this could be the height of a ball falling down).

Independent variable: t (time is what we track to see how the height y changes).

In short:

  • Dependent variable: The quantity whose value depends on other variables (e.g., y, u).
  • Independent variable: The variable that the dependent variable changes with respect to (e.g., t, x).

Classifications of Differential Equations

Let's go back to the toy car. Now, imagine we have some rules about how it moves. These rules (or equations) can be classified in three ways:

1 By Order:

The order of a differential equation is given by the highest-order derivative

    First Order: This deals with how something is changing (like speed) but only looks at the first change.

  • Example: \( \frac{dy}{dt} = 5y \)

    • The speed (y) changes based on how fast it’s moving over time.

    Second Order: This looks at how both the speed and the rate of speed change (like acceleration).

  • Example: \( \frac{d2y}{dt2} = -9.8 \)

    • This could describe something like gravity pulling a ball down, where the second derivative represents how fast the speed is changing (acceleration).

2.By Type:

    Ordinary Differential Equation (ODE): This type deals with only one variable changing over time.

  • Example: \( \frac{dy}{dt} = 3t \)

    • This means something (let's say the speed of the toy car, y) is changing based on time t.

    Partial Differential Equation (PDE): This type involves more than one variable changing, like time and space (imagine the car moving both in time and along a track with hills).

  • Example: \( \frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2} \)

    • Here, u could be how the height of the car on a hill changes with both time t and position x.

3.By Linearity:

  • Linear: These equations are simple and follow straight rules (no loops or curves).

  • Example:\( \frac{dy}{dt} + 2y = 4 \)

    • This means that y changes smoothly without complicated behavior.

    Nonlinear: These are more complex, with twists and turns, like a rollercoaster.

  • Example:\( \frac{dy}{dt} = y^2 -t \)

    • This rule has the variable (y) affecting itself in a squared way, making the changes more unpredictable.

A differential equation is considered linear if it can be written in the following form:

Equation: \( a_n(x) \frac{d^n y}{dx^n} + a_{n-1}(x) \frac{d^{n-1} y}{dx^{n-1}} + \cdots + a_1(x) \frac{dy}{dx} + a_0(x)y = f(x) \)

So, these are ways to organize the rules that describe how things (like the car) move and change!

Explanation of Linear Differential Equation

y is the dependent variable, and x is the independent variable.

\( a_n(x), a_{n-1}(x), \cdots, a_0(x) \) are functions of x (or can be constants).

f(x) is some function of x, often called the "forcing function."

The equation is linear in terms of y, which means no powers of y greater than 1 and no products of derivatives of y are involved.

Example of a Linear Equation:

Equation:

\[ x^2 \frac{d^2y}{dx^2} + 3x \frac{dy}{dx} + 2y = \sin(x) \]

Here:

  • \( a_2 = 1 \) (coefficient of \( \frac{d^2y}{dx^2} \))
  • \( a_1 = 3 \) (coefficient of \( \frac{dy}{dx} \))
  • \( a_0 = 2 \) (coefficient of \( y \))
  • f(x) = sin(x) is the forcing function.

This is a linear equation because y and its derivatives appear to the first power only, and no products of y or its derivatives appear.

Nonlinear Differential Equation

A differential equation is nonlinear if it cannot be written in the linear form described above. This can happen if:

  • The dependent variable y (or its derivatives) is raised to a power other than 1.
  • Products of y and its derivatives appear.
  • Functions like \(\sin(y)\) or \(\exp(y)\) are involved.

Example of a Nonlinear Equation:

Equation:

\[ \frac{d^2y}{dx^2} + \sin(y) = 0 \]

In this case, \(\sin(y)\) makes the equation nonlinear because the sine function involves the dependent variable \(y\).

More Examples:

Linear:

\[ y'' + 5y' - 4y = \cos(x) \]

Nonlinear:

\[ y'' + y^2 = 0 \]

Key Differences:

  • In a linear equation, the dependent variable y and all its derivatives appear linearly (i.e., to the power of 1).
  • In a nonlinear equation, y (or its derivatives) appear raised to a power, multiplied together, or involved in other non-linear operations like sine or exponential functions.

Understanding Differential Equations: The Magic Box Analogy

At this point, it is useful to highlight certain properties that all differential equations share, regardless of their type, order, and whether they are linear or not.

First, it is important to note that just because we can write down a differential equation, it doesn’t mean that a solution exists. The existence of solutions for a class of differential equations is a key aspect of the theory of differential equations. Since we are interested in differential equations that arise from real-world applications, we need to ensure that their solutions actually exist.

What Happens When We Use the Box?

Magic Box: Imagine we have a magic box called a differential equation. This box is special because it helps us understand how things change, like the motion of a toy car or the ascent of a balloon in the air.

When We Put Something In: When we place something into the magic box (like a number or an equation), we expect it to return a useful answer, such as where the car will be after some time.

Does It Always Work? Here’s the tricky part: just because we have a magic box doesn’t mean it always works perfectly. Sometimes, when we put something in, the box might not give us any answer at all. This can be confusing and frustrating!

Finding Solutions: When we talk about wanting a solution, we mean we want to know what happens next. For example, if we push our toy car, we want to determine where it will go afterward. For our magic box to be truly useful, it should always provide an answer when we use it.

Why This is Important:

In real life, when we seek to understand how things change (like the height of a tree as it grows), we need to be confident that our magic box can help us find that answer. If it doesn’t, we cannot use it effectively to make sense of the world around us.

Summary:

  • Magic Box (Differential Equation): Helps us figure out how things change over time.
  • Expect Answers (Solutions): We want to know the outcomes of our inputs.
  • Importance: The magic box should consistently provide answers to ensure we can understand our world better.

Types of Solutions in Differential Equations

Consider the differential equation:

\( \frac{dy}{dx} = x + 1 \)

This equation means that the rate of change of \( y \) with respect to \( x \) is given by \( x + 1 \). This tells us that for every increase in \( x \), \( y \) increases by \( x + 1 \).

Initial Condition

The initial condition \( y(1) = 2 \) tells us that when \( x = 1 \), the value of \( y \) is 2. This helps us find a specific solution to the equation.

Speaking of Solutions, Here Are the Different Types:

  • 1. General Solution:

    The general solution of a differential equation includes an arbitrary constant because it represents a family of possible solutions. It reflects the overall behavior of the system without focusing on specific values.

    In the case of \( \frac{dy}{dx} = x + 1 \), the general solution is:

    \( y = \frac{x^2}{2} + x + C \)

    Here, \( C \) is the arbitrary constant that can take any value, depending on the initial conditions.

  • 2. Particular Solution:

    A particular solution is obtained by applying the initial condition (or boundary conditions) to the general solution. This provides a specific solution that satisfies both the equation and the initial condition.

    In our case, using \( y(1) = 2 \), we substitute \( x = 1 \) and \( y = 2 \) into the general solution to solve for \( C \):

    \( 2 = \frac{1^2}{2} + 1 + C \)

    Simplifying gives \( C = \frac{1}{2} \), so the particular solution is:

    \( y = \frac{x^2}{2} + x + \frac{1}{2} \)

  • 3. Singular Solution:

    A singular solution is a solution that cannot be obtained from the general solution by choosing any value of the constant \( C \). These solutions are often derived from special conditions in the differential equation itself and are unique in nature.

    In some cases, differential equations can have a singular solution that satisfies the equation in a way that no other solution can.

  • 4. Implicit Solution:

    An implicit solution is one where the solution for \( y \) is not explicitly given as a function of \( x \), but rather in a form that involves both \( y \) and \( x \). To find an explicit solution, additional steps are usually needed.

    For example, an implicit solution might look like:

    \( F(x, y) = 0 \)

    This form may still provide valuable information about the behavior of the system, even if it's not expressed in explicit terms.

  • 5. Numerical Solution:

    When an exact or analytic solution is difficult to find, numerical methods are often used. These methods approximate the solution using computers and algorithms, such as Euler's method or Runge-Kutta methods. This is especially useful for complex equations where exact solutions are not feasible.

Summary

When solving a differential equation, there are different types of solutions, including the general, particular, singular, implicit, and numerical solutions. Each type serves a specific purpose in helping us understand how systems change over time.

Understanding the initial condition helps us narrow down the family of solutions to one particular solution that applies to a specific real-world scenario.

Consider the Differential Equation:

\( \frac{dy}{dx} = x + 1 \)

This equation means that the rate of change of \( y \) with respect to \( x \) is given by \( x + 1 \). In simple terms, for every increase in \( x \), \( y \) increases by \( x + 1 \).

Initial Condition:

\( y(1) = 2 \)

This initial condition tells us that when \( x = 1 \), the value of \( y \) is 2. This information will help us find the specific solution to the differential equation.

Now, we can use the initial condition to find the constant of integration in the solution process.

Step 1: Solve the Differential Equation

To solve the equation \( \frac{dy}{dx} = x + 1 \), we can integrate both sides:

  • \( y = \int (x + 1) \, dx = \frac{x^2}{2} + x + C \)

Where \( C \) is the integration constant. Now, we use the initial condition \( y(1) = 2 \) to find \( C \):

\( 2 = \frac{1^2}{2} + 1 + C \)

Simplifying:

\( 2 = \frac{1}{2} + 1 + C \)

\( 2 = \frac{3}{2} + C \)

So, we solve for \( C \):

\( C = 2 - \frac{3}{2} = \frac{4}{2} - \frac{3}{2} = \frac{1}{2} \)

Thus, the particular solution is:

\( y = \frac{x^2}{2} + x + \frac{1}{2} \)

Step 2: Implementing in MATLAB/GNU OCTAVE

Here’s how you can implement this in MATLAB to visualize the solution:


% Define the function as an anonymous function
f = @(x) (x.^2)/2 + x + 1/2;
            
% Create a range of x values
x = linspace(0, 3, 100);  % From x = 0 to x = 3
            
% Calculate corresponding y values using the function
y = f(x);
            
% Plot the results
figure;
plot(x, y, 'b-', 'LineWidth', 2);
hold on;
plot(1, 2, 'ro', 'MarkerSize', 8); % Initial condition point
text(1, 2, ' (1, 2)', 'VerticalAlignment', 'bottom', 'HorizontalAlignment', 'right');
xlabel('x');
ylabel('y');
title('Solution of the Differential Equation dy/dx = x + 1');
grid on;
legend('y = (x^2)/2 + x + 1/2', 'Initial Condition (1, 2)');
hold off;

Step 3: Implementing in Python

Now, let’s implement the same solution in Python using numpy and matplotlib.


import numpy as np
import matplotlib.pyplot as plt

# Define the function for the solution
def f(x):
    return (x**2) / 2 + x + 1 / 2

# Create a range of x values
x = np.linspace(0, 3, 100)  # From x = 0 to x = 3

# Calculate corresponding y values using the function
y = f(x)

# Plot the results
plt.figure(figsize=(8, 6))
plt.plot(x, y, 'b-', linewidth=2, label='y = (x^2)/2 + x + 1/2')
plt.plot(1, 2, 'ro', markersize=8)  # Initial condition point
plt.text(1, 2, ' (1, 2)', verticalalignment='bottom', horizontalalignment='right')
plt.xlabel('x')
plt.ylabel('y')
plt.title('Solution of the Differential Equation dy/dx = x + 1')
plt.grid()
plt.legend()
                    
plt.show()
