🔹 Why rsync is Better than mv

✔ Verifies file integrity (uses checksums to ensure no corruption).
✔ Faster (only moves changed files instead of everything).
✔ Resumable (if the transfer is interrupted, you can restart it without losing progress).
✔ Safer (preserves permissions, ownership, timestamps).

🔹 Using rsync to Move Docker Data

Use:

  • -X → Preserves extended attributes.
  • -v → Verbose mode (shows progress).
  • --progress → Displays real-time progress of the transfer.
